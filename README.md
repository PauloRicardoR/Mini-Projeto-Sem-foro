# Semáforo

Este é um projeto de simulação de semáforo desenvolvido usando HTML, CSS e JavaScript. 
O semáforo permite que os usuários alternem entre os estados (vermelho, amarelo, verde) manualmente ou automaticamente.

## Como usar

1. Abra o arquivo `index.html` em um navegador web.
2. Clique nos botões `Vermelho`, `Amarelo` ou `Verde` para alternar manualmente o estado do semáforo.
3. Clique no botão `Automático` para iniciar a alternância automática dos estados do semáforo.

## Estrutura dos arquivos

- `index.html`: Contém a estrutura HTML do semáforo.
- `style.css`: Estiliza a aparência do semáforo.
- `script.js`: Contém a lógica JavaScript para alternar os estados do semáforo.
